## 🐍 What is this?

A **self-contained, single-file exam simulator** built from a 508+ question Python MCQ bank — designed for developers preparing for technical interviews at TCS, MNCs, and senior engineering roles.

No frameworks. No server. No login. Just open and practice.

---

## ✨ Features

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   🟢  EASY    →  150 questions  │  Syntax, builtins     │
│   🟡  MEDIUM  →  168 questions  │  OOP, decorators      │
│   🔴  HARD    →  165 questions  │  Internals, async     │
│   ⚡  MIX     →   50 questions  │  Random every time    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

| Feature | Details |
|---|---|
| ✅ Correct answer | Highlights green + shows explanation |
| ❌ Wrong answer | Subtle red flash — keep trying |
| ⚡ Mix Mode | 50 dynamically shuffled questions per session |
| 🔁 Reshuffle | Get a fresh 50 with one click |
| 📊 Live score | Correct count updates as you go |
| 📱 Responsive | Works on mobile & desktop |
| 🌙 Dark theme | Easy on the eyes for long sessions |

## 🚀 How to Use

**Option 1 — Live on GitHub Pages:**
```
https://gerwin-demo.github.io/P-for-Python---MCQ-INTERVIEW-PREP/
```

**Option 2 — Run locally:**
```bash
# Clone the repo
git clone https://github.com/gerwin-demo/P-for-Python---MCQ-INTERVIEW-PREP.git

# Open in browser — no install needed
open index.html
```
